# Topic 9: AI & Neural Networks (35 Q)

<details>
<summary><b>Q: Artificial Intelligence definition?</b></summary>
<blockquote><b>Answer:</b> Creating intelligent machines that perform tasks requiring human-like intelligence</blockquote>
</details>

<details>
<summary><b>Q: Intelligent Agent?</b></summary>
<blockquote><b>Answer:</b> Perceives environment, takes actions to achieve goals</blockquote>
</details>

<details>
<summary><b>Q: State Space Search?</b></summary>
<blockquote><b>Answer:</b> Represents problem as states and transitions. Find path from initial to goal</blockquote>
</details>

<details>
<summary><b>Q: Machine Learning types?</b></summary>
<blockquote><b>Answer:</b> Supervised (labeled), Unsupervised (unlabeled), Reinforcement (reward-based)</blockquote>
</details>

<details>
<summary><b>Q: Classification vs Regression?</b></summary>
<blockquote><b>Answer:</b> Classification: discrete output (category). Regression: continuous output (number)</blockquote>
</details>

<details>
<summary><b>Q: Decision Tree?</b></summary>
<blockquote><b>Answer:</b> Tree-based classifier. Each node tests feature, leaves are predictions</blockquote>
</details>

<details>
<summary><b>Q: Naive Bayes?</b></summary>
<blockquote><b>Answer:</b> Probabilistic classifier. Assumes feature independence. P(class|features)</blockquote>
</details>

<details>
<summary><b>Q: K-Means clustering?</b></summary>
<blockquote><b>Answer:</b> Unsupervised. Partitions data into K clusters. Centroid-based</blockquote>
</details>

<details>
<summary><b>Q: Neural Network?</b></summary>
<blockquote><b>Answer:</b> Inspired by brain neurons. Interconnected nodes (neurons) in layers</blockquote>
</details>

<details>
<summary><b>Q: Perceptron?</b></summary>
<blockquote><b>Answer:</b> Simplest NN. Single layer. Binary classification</blockquote>
</details>

<details>
<summary><b>Q: Sigmoid function range?</b></summary>
<blockquote><b>Answer:</b> Output: 0 to 1. Used in binary classification</blockquote>
</details>

<details>
<summary><b>Q: ReLU (Rectified Linear Unit)?</b></summary>
<blockquote><b>Answer:</b> max(0, x). Fast, prevents vanishing gradient</blockquote>
</details>

<details>
<summary><b>Q: Tanh function range?</b></summary>
<blockquote><b>Answer:</b> Output: -1 to 1</blockquote>
</details>

<details>
<summary><b>Q: Softmax function?</b></summary>
<blockquote><b>Answer:</b> Converts scores to probabilities. Multi-class classification</blockquote>
</details>

<details>
<summary><b>Q: Backpropagation?</b></summary>
<blockquote><b>Answer:</b> Algorithm to train NN. Computes gradients, updates weights to minimize error</blockquote>
</details>

<details>
<summary><b>Q: Gradient Descent?</b></summary>
<blockquote><b>Answer:</b> Optimization algorithm. Moves towards minimum error</blockquote>
</details>

<details>
<summary><b>Q: Learning rate?</b></summary>
<blockquote><b>Answer:</b> Step size in gradient descent. Too high: miss optimum. Too low: slow</blockquote>
</details>

<details>
<summary><b>Q: Overfitting?</b></summary>
<blockquote><b>Answer:</b> Model learns noise, not pattern. Good training, poor test accuracy</blockquote>
</details>

<details>
<summary><b>Q: Underfitting?</b></summary>
<blockquote><b>Answer:</b> Model too simple, misses pattern. Poor both train and test</blockquote>
</details>

<details>
<summary><b>Q: Regularization?</b></summary>
<blockquote><b>Answer:</b> Prevents overfitting. L1/L2 penalty on weights</blockquote>
</details>

<details>
<summary><b>Q: Dropout?</b></summary>
<blockquote><b>Answer:</b> Randomly disable neurons during training. Prevents co-adaptation</blockquote>
</details>

<details>
<summary><b>Q: Batch Normalization?</b></summary>
<blockquote><b>Answer:</b> Normalizes layer inputs. Stabilizes training, allows higher learning rates</blockquote>
</details>

<details>
<summary><b>Q: CNN (Convolutional Neural Network)?</b></summary>
<blockquote><b>Answer:</b> Uses convolutional layers. Effective for images. Weight sharing</blockquote>
</details>

<details>
<summary><b>Q: RNN (Recurrent Neural Network)?</b></summary>
<blockquote><b>Answer:</b> Has feedback connections. Processes sequences. Memory of past</blockquote>
</details>

<details>
<summary><b>Q: LSTM (Long Short-Term Memory)?</b></summary>
<blockquote><b>Answer:</b> Type of RNN. Solves vanishing gradient. Memory cells</blockquote>
</details>

<details>
<summary><b>Q: Genetic Algorithm?</b></summary>
<blockquote><b>Answer:</b> Mimics evolution. Selection, crossover, mutation. Population-based optimization</blockquote>
</details>

<details>
<summary><b>Q: Reinforcement Learning?</b></summary>
<blockquote><b>Answer:</b> Agent learns via reward/penalty. Trial and error. Q-learning, Policy gradient</blockquote>
</details>

<details>
<summary><b>Q: NLP (Natural Language Processing)?</b></summary>
<blockquote><b>Answer:</b> Understanding/generating human language. Tokenization, POS tagging, sentiment analysis</blockquote>
</details>

<details>
<summary><b>Q: Computer Vision?</b></summary>
<blockquote><b>Answer:</b> Understanding images. Object detection, classification, segmentation</blockquote>
</details>

<details>
<summary><b>Q: Feature Engineering?</b></summary>
<blockquote><b>Answer:</b> Creating relevant features from raw data. Critical for model performance</blockquote>
</details>

<details>
<summary><b>Q: Cross-validation?</b></summary>
<blockquote><b>Answer:</b> Splits data into folds. Trains on k-1, tests on 1. Reduces overfitting</blockquote>
</details>
